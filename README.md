# Steps

## Step 1 : Clone Repo

## Step 2 :
```bash
cd frontend
npm i
npm run dev
```

## Step 3 :
login to MySQL shell
```bash
source ./backend/database/schema.sql;
```
and then

add MySQL userid and password in
*backend/.env* file

## Step 4 :
```bash
cd backend
npm i
npm run dev
```
